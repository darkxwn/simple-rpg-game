\## 🇬🇧 English



\### 📖 About

This is a simple \*\*educational project\*\* created to study game development mechanics using C++ and the SFML library. The primary goal is learning modern C++ practices and graphics programming, so the code structure and gameplay are kept straightforward.



\### 🛠 Technology Stack

\- \*\*Language:\*\* C++ (Standard \*\*C++17\*\* or higher required)

\- \*\*Graphics Library:\*\* \*\*SFML 3.0.2\*\* (Simple and Fast Multimedia Library)

\- \*\*Build System:\*\* CMake (recommended) or manual Makefile/VS Project



\### 🎨 Graphics \& Assets

All sprites, textures, and graphical assets used in this project are taken from public resources on the internet. They are used solely for \*\*educational purposes\*\* within this non-commercial project.
